iReport的使用

iReport与JasperReport简介

JasperReport是报表的引擎部分，界面设计是用iReport。

iReport的文件分为两种，一种为.jrxml源文件（类似.java），一种为.jasper编译文件（类似.class）。

注:在ireport里拖拽生成的.jrxml文件，浏览的为.jasper文件。

iReport中制作报表常用的组件

Rectangle:用于画表格的样式，本控件表现为一个黑矩形框。

Static Text:静态文本框，报表上显示的静态文本如标题、表头等

Text Field:报表中最常用的控件，呈现数据库字段的值

Subreport:子报表（用于制作复杂的报表）

1.根据报表用Rectangle控件拼出表结构

报表主要包括标题、表头和数据部分，单位所在行是表头部分。标题和表头放在PageHeader区内；下面是数据detail部分，数据detail部分显示的行是由数据库的记录条数决定的（只需要在detail里面设计一个行模板）

2.使用Static Text加标题和表头信息

Static Text组件常用的属性:

Text：设置显示的文本（注：需要输入回车符，需要点击后面的省略号，在弹出的对话框中设置）

Font name：设置字体

Size：设置字体大小

Horizontal Alignment：报表水平对齐方式

Vertical Alignment：报表垂直显示方式

以下两个属性是输出PDF格式时必须设置的，否则中文不显示。

PDF Font name：设置成STSong-Light

PDF Encoding：设置成UniGB-UCS2-H(中文)

3.使用Text Field将数据库里的数据显示到报表中

A.连接数据库

选择数据源类型、配置连接数据源信息

注：红色代表是未安装的驱动

B.指定数据源

数据库连接成功后，为报表指定数据源，在report query对话框中制定查询（将查询的数据对应显示在detail区中）

C.将数据库中的字段添加到报表中

D.Text Field常用属性介绍

Print When Expression：Text Field组件显示条件设置。

Blank When Null：此属性一般都需要选择（避免没有数据情况下报表中显示null值）
